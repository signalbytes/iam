The 'Perfect Paragraph' Prompt: How to Make ChatGPT Sound Like You

We've all read them. Those overly polished, soulless paragraphs that scream "I was written by an AI." They use words like "delve," "tapestry," and "moreover" in every other sentence.

As a writer and developer, I refuse to publish generic AI slop. But I also refuse to ignore a tool that can 10x my drafting speed. The secret isn't in the editing; it's in the prompting.

Stop Asking for "Articles"

When you ask an LLM to "write an article," it defaults to its training data average—which is a bland, academic essay format. It writes to be safe, not to be interesting.

Instead, you need to constrain the AI. You need to give it a strict persona, a specific tone, and a list of forbidden words.

The Role-Playing Framework

The best way to get human-sounding text is to make the AI role-play as a specific type of writer. I usually tell it to act like a senior tech journalist writing for a cynical audience.

  • Tone: Direct, punchy, and slightly conversational.
  • Structure: Short paragraphs. No fluff.
  • Constraints: Ban the typical AI vocabulary.

The Master Prompt

Copy and paste this exact prompt before you ask the AI to write anything. It will fundamentally change the output quality.

You are a senior tech journalist writing for an expert audience. 
Write a 300-word explanation of [TOPIC].

Follow these strict rules:
1. Use a first-person, conversational tone.
2. Keep paragraphs short (maximum 3 sentences).
3. DO NOT use the following words: delve, tapestry, moreover, furthermore, landscape, crucial, vital.
4. Start directly with the main point. No generic introductions.
5. Use active voice and punchy verbs.
